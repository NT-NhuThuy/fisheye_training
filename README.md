# Fisheye Object Detection Training

Dự án này chứa các script để xử lý dữ liệu và huấn luyện mô hình YOLO trên tập dữ liệu fisheye.

## 🛠 Cài đặt
```bash
pip install -r requirements.txt
```

## 📈 Hướng dẫn sử dụng
- Chia dữ liệu: `python scripts/split_dataset.py`
- Training: `python scripts/train_yolo26.py`
