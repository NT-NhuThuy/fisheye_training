# Fisheye Training

This repository contains two main parts:
- `training/`: dataset splitting and object detection training
- `violation_speedlimit/`: speed violation inference pipeline for fisheye traffic videos

## Installation
```bash
pip install -r requirements/training.txt
pip install -r requirements/inference.txt
```

## Quick Start
- Create your own dataset config from `training/configs/dataset.example.yaml`, or run:
  `python training/split_dataset.py --data-dir /path/to/data_fisheye`
- Train YOLO11: `python training/scripts/train_yolo11.py`
- Train YOLO26: `python training/scripts/train_yolo26.py`
- Train YOLOv8: `python training/scripts/train_yolov8.py`
- Train RT-DETR: `python training/scripts/train_rtdetr.py`
- Run inference: `python violation_speedlimit/src/main.py --source path/to/video.mp4`

## Training Notes
- Training scripts will use `training/configs/dataset_balanced.yaml` first if it has been generated by `split_dataset.py`.
- You can choose a GPU with the `YOLO_DEVICE` environment variable.
- You can override the pretrained weight path with `YOLO_MODEL_PATH`.
